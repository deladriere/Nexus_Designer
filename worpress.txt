Nexus vox Designer is a single‑file, visual app that runs directly in Chromium‑based browsers (Chrome, Edge, Brave, Opera). No install, no drivers — open the file and connect.

<samp>Quick start</samp>
1) Open <code>Vax_vox_designer.html</code> in a Chromium‑based browser (drag the file into a tab or use File → Open).
2) Click <strong>Connect</strong> and pick your <a href="https://www.polaxis.be/product/kraftor/">Kraftor</a> (hosting<a href="https://www.polaxis.be/product/vax-vox/"> Vax vox </a>and running<a href="https://www.polaxis.be/vax-serial-midi/">  vax-serial-midi</a>.
3) Tweak the <strong>Voice tuning</strong> section (Name, Pitch, Breathiness, Head size, Laryngealization, Gain). The command field below updates automatically.
4) Type text and press <strong>Speak</strong>, or use the MIDI pads to trigger notes 36–47.

<samp>What’s inside</samp>
- Voice tuning dials with tooltips
- Editable <strong>Designed voice command</strong> field (auto‑updates and debounced auto‑sends <code>/d …</code>)
 - Editable <strong>Designed voice command</strong> field (auto‑updates and debounced auto‑sends <code>/d …</code>) — accepts any DECtalk command, not just the dials (including hidden gems like <code>g5</code> to tame clipping)
- Reset sets sensible defaults and sends <code>/o</code>
- Voice selection (radio) sends <code>/v{index}</code>
- Collapsible Voice tuning section with chevron (▼/▶) to save vertical space
- MIDI note pads (36–47) can be used to trigger sentences from the browser, as well as from your favorite MIDI application.

<samp>Presets</samp>
- Save presets locally in your browser (per profile/device)
- Select a preset to load it instantly (auto‑applies command)
- Export/Import presets as JSON to share or back them up

<samp>Device &amp; connections</samp>
- Device name shown on connect (via WebUSB when available)
- Web Serial &amp; Web MIDI; status badges for each connection

<samp>Storage &amp; privacy</samp>
- Everything is stored locally in your browser (LocalStorage). Runs offline; data stays on your machine.

<samp>Compatibility</samp>
- Desktop: Chrome/Chromium 89+ (macOS / Windows / Linux)
- Not supported: iOS/iPadOS Safari (no Web Serial), most mobile browsers

<samp>Tips</samp>
- Hover the labels to see tooltips
- If the device doesn’t appear, close other serial apps that might hold the port
- If the browser says Web Serial isn’t supported, update Chrome/Edge/Brave on desktop

<samp>Known behavior</samp>
- The browser may ask for permission to read friendly USB names (WebUSB)

<samp>Support</samp>
Include your OS, browser version, and a short description of your device/baud settings when reporting issues.
